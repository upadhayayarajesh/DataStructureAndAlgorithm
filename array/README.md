# Operations on Array Time complexity

## Insert operation

- Search
    - Unsored array
        + o(n)
    - Sorted
        - O(logN)
- Insertion
    + Beginning of the list
        + O(N)
    + End of the list
        + O(1)
- Delete
    - O(N)
- Get i-th Element
    - O(1)
- Update i-th Element
    - O(1)

> [!NOTE]
> Insert at the end and delete from the end can be done in O(1) time.
